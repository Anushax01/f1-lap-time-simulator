# F1 2026 Lap Time Simulator

A MATLAB simulation that predicts theoretical lap times for all Formula 1 2026 circuits using vehicle dynamics modeling.

## Features
- Simulates all 24 F1 2026 Grand Prix circuits
- Calculates speed profiles, top speeds, and cornering limits
- Pit stop strategy optimization (1-stop, 2-stop, etc.)
- Weather impact analysis (dry, damp, wet, heavy rain)
- Visual output with speed profiles and track representations

## How to Use
1. Open the MATLAB script `lap_time_2026_all_tracks.m`
2. Run the code (press F5)
3. Enter a circuit number (1-24) when prompted
4. View results in the command window and figure window

## Example Results (Mexican GP)
- Top Speed: ~371 km/h
- Estimated Lap Time (sector): ~33.0 seconds
- Best Strategy: 2-stop pit strategy

## Requirements
- MATLAB (any recent version)
